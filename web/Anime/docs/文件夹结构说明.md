## 目录职责

- `app.py`
  Python 后端入口，负责初始化 SQLite、路由分发、模板渲染与静态资源服务。
- `data/anime.db`
  SQLite 数据库，存放番剧标题、简介、演员、关键词、海报路径、剧照路径和资料来源。
- `poster/`
  首页海报墙使用的竖版海报资源。
- `Stills/`
  详情页使用的横版背景剧照资源。
- `templates/`
  页面模板，包含首页和详情页的 HTML。
- `static/`
  样式与前端交互脚本，负责海报墙视觉风格、详情页布局、搜索筛选与预取优化。
- `docs/`
  项目结构、文本流程图和资料来源说明。
