## 目录职责

- `app.py`
  Python 后端入口，负责初始化 SQLite、路由分发、模板渲染、播放地址保存、剧集配置保存和剧集跳转。
- `README.md`
  项目总说明，包含功能概览、运行方式、目录说明和使用声明。
- `data/anime.db`
  SQLite 数据库，存放番剧标题、简介、演员、关键词、海报路径、剧照路径、播放地址、剧集配置和上一次播放记录。
- `poster/`
  首页海报墙使用的竖版海报资源。
- `Stills/`
  详情页使用的横版背景剧照资源。
- `templates/`
  页面模板，包含首页和详情页 HTML。
- `static/`
  样式与前端交互脚本，负责海报墙视觉风格、详情页布局、搜索筛选、播放地址编辑模式和剧集配置展开逻辑。
- `docs/`
  项目结构、流程、运行、移植和资料来源说明。
- `prompt.md`
  初始任务要求记录文件。
