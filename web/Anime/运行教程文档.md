# 运行教程文档

## 环境要求

- 操作系统：Windows 或 Ubuntu
- 浏览器：Google Chrome
- Python：3.10 及以上

## 启动步骤

在项目根目录执行：

```bash
python3 app.py
```

默认启动后会监听：

```text
http://127.0.0.1:8000
```

## 首次运行会发生什么

- 自动创建 `data/anime.db`
- 自动写入番剧基础数据
- 自动加载 `poster/` 和 `Stills/` 中的图片资源

## 指定端口启动

```bash
python3 app.py --port 8080
```

## 仅初始化数据库

```bash
python3 app.py --init-db
```

## 页面使用说明

- 首页展示番剧海报墙
- 点击任意海报可进入详情页
- 首页右上角搜索框可即时筛选番剧
- 详情页展示简介、剧集演员、关键词和资料来源

## 常见问题

### 1. 浏览器无法打开页面

- 检查终端是否显示 `Serving on http://127.0.0.1:8000`
- 检查端口是否被其他程序占用

### 2. 图片不显示

- 确认 `poster/` 和 `Stills/` 目录仍然存在
- 确认图片文件名没有被改动

### 3. 想更新番剧内容

- 修改 `app.py` 中的 `CATALOG_SEED`
- 运行 `python3 app.py --init-db`
- 重新启动网页服务
