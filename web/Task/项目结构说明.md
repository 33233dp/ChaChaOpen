# 项目结构说明

```text
Task/
├── app.py
├── index.html
├── heatmap.html
├── webProgramming.md
├── 运行教程文档.md
├── 代码复制移植说明书.md
├── 流程图说明.md
└── 项目结构说明.md
```

## 说明

- `app.py` 是 Python 后端，负责提供 API、初始化 SQLite 数据库、托管首页。
- `index.html` 是任务页，负责 UI 渲染、交互和调用后端接口。
- `heatmap.html` 是独立的年执行热力图页，只负责展示年度任务操作密度。
- `task_manager.db` 会在第一次启动时自动生成，用于持久化任务数据和任务操作日志。
- `运行教程文档.md` 说明如何启动和访问应用。
- `代码复制移植说明书.md` 说明如何把整个项目搬到其他机器。
- `流程图说明.md` 描述前端和后端的调用关系。
