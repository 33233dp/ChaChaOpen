# 运行教程文档

## 环境要求

- Python 3.10+
- Windows 或 Linux
- Google Chrome 或其他现代浏览器

## 一行启动

在当前目录直接执行：

```bash
python app.py
```

如果系统没有 `python` 命令，可以改用：

```bash
python3 app.py
````

## 启动后访问

打开浏览器并访问：

```text
http://127.0.0.1:8000/
```

任务页和热力图页已经拆开：

- 任务页：`http://127.0.0.1:8000/`
- 热力图页：`http://127.0.0.1:8000/heatmap`

## 功能使用

1. 在“新建任务”区域输入标题、备注、截止日期和优先级。
2. 点击“添加任务”保存到 SQLite 数据库。
3. 在任务列表中点击圆点切换完成状态。
4. 点击“编辑”修改任务。
5. 点击“删除”移除任务。
6. 使用左侧筛选和顶部搜索快速定位任务。
7. 点击“查看热力图”进入独立的年执行热力图页面。

## 停止服务

- 在终端按 `Ctrl + C`。

## 常见问题

### 页面打不开

- 确认终端里已经打印出启动地址。
- 确认 8000 端口没有被其他程序占用。

### 数据没有保存

- 检查当前目录下是否生成了 `task_manager.db`。
- 确认程序启动和退出时没有报错。

### 想换端口

```bash
PORT=8080 python app.py
```

如果你的系统使用 `python3`，就把上面的命令改成 `PORT=8080 python3 app.py`。
