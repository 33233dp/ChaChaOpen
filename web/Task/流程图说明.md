# 流程图说明

```text
浏览器访问 `http://127.0.0.1:8000/`
        │
        ▼
`app.py` 中的 `TodoHandler.do_GET()`
        │
        ├── 路径 `/` 或 `/index.html`
        │     ▼
        │   读取任务页 `index.html`
        │     ▼
        │   浏览器加载任务页
        │
        ├── 路径 `/heatmap` 或 `/heatmap.html`
        │     ▼
        │   读取热力图页 `heatmap.html`
        │     ▼
        │   浏览器加载独立热力图页
        │
        ├── 路径 `/api/tasks`
        │     ▼
        │   `do_GET()` → 查询 SQLite
        │     ▼
        │   `row_to_task()` 转换数据
        │     ▼
        │   返回 JSON 给前端
        │
        ├── 路径 `/api/stats`
        │     ▼
        │   统计任务数量
        │     ▼
        │   返回 JSON 给前端

        └── 路径 `/api/activity`
              ▼
            聚合 `task_events` 表中的年度操作次数
              ▼
            返回热力图数据给前端

前端交互流程

用户填写新任务表单
        │
        ▼
`index.html` 中的 `createTask()`
        │
        ▼
调用 `POST /api/tasks`
        │
        ▼
`TodoHandler.do_POST()`
        │
        ▼
`validate_task_payload()`
        │
        ▼
写入 `task_manager.db`
        │
        ▼
同步写入 `task_events` 任务操作日志
        │
        ▼
前端重新调用 `loadDashboard()`

独立热力图页流程

用户点击任务页上的“查看热力图”
        │
        ▼
打开 `/heatmap`
        │
        ▼
`heatmap.html` 中的 `loadHeatmap()`
        │
        ▼
调用 `GET /api/activity?year=YYYY`
        │
        ▼
`TodoHandler.do_GET()`
        │
        ▼
聚合 `task_events` 表并返回年度热力图数据

任务完成/编辑/删除流程

用户点击复选框或操作按钮
        │
        ├── `toggleTask()`
        │     ▼
        │   `PATCH /api/tasks/<id>/toggle`
        │     ▼
        │   `TodoHandler.do_PATCH()`
        │     ▼
        │   更新 `completed`
        │
        ├── `saveTask()`
        │     ▼
        │   `PATCH /api/tasks/<id>`
        │     ▼
        │   `validate_task_payload()`
        │     ▼
        │   更新任务字段
        │
        └── `deleteTask()`
              ▼
            `DELETE /api/tasks/<id>`
              ▼
            从数据库移除记录
```
