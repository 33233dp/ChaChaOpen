拉格朗如函数
L = K - P
拉格朗日函数
![[Pasted image 20260508102836.png]]
代表第i个广义坐标的非保守广义力
注意F_i不是外力，而是广义力，广义力中还包含阻尼力
>**阻尼力**就是阻碍相对运动的力，通常和速度有关。
>**广义力**是广义坐标方向上的等效力

拉格朗日函数是F = ma的广义形式
比如普通直线运动:
$K=\frac12m\dot x^2$
没有势能时： 
$L=K$
代入拉格朗日方程：
$F=\frac{d}{dt}\frac{\partial L}{\partial \dot x}-\frac{\partial L}{\partial x}$  
右边变成：
$\frac{d}{dt}(m\dot x)-0=m\ddot x$ 
所以：
$F=m\ddot x$ 
这就是牛顿第二定律。
如果有弹簧势能：
$P=\frac12kx^2$  
那么：
$L=K-P$  
代入后得到：
$F=m\ddot x+kx$  
整理为：
$m\ddot x+kx=F$  
这就是带弹簧的动力学方程。
所以你可以记成：
$\boxed{\text{拉格朗日方程 }=\text{ 用能量写出来的 }F=ma}$  

注意F_i不是外力，是物体受到的广义力，这个力由外力以及阻尼器提供，如：
![[Pasted image 20260508110303.png]]
上面那个是外力，下面这个是由阻尼器提供。

**总而言之，拉格朗日方程描述的是力与运动关系的方程**
**本质上来说就是：**
>外部施加/耗散的力 = 惯性项 + 势能产生的保守力项
>惯性项：质量乘以加速度，来自动能K
>保守力项：由势能对位移求导得到的力
>非保守力：不能由势能对位移求导得到。
>势能：与位置，形变，构型有关，可以存储的量。
>阻尼器：耗散能量，不能存储，把机械能变成热能。

# 代码实践
![[Pasted image 20260508133659.png]]
``` matlab
% 《机器人学 第四版 清华大学出版社》 p117 4.1.1 刚体的动能与位能

syms t;

syms K;

syms x0(t) x1(t);

syms M0 M1;

  

syms k;

syms g;

syms c;

  

syms F;

  
  

dx0 = diff(x0,t);

dx1 = diff(x1,t);

  

K = 1/2 * M0 * dx0^2 + 1/2 * M1 * dx1^2;    % 动能

P = 1/2 * k * (x1 - x0)^2 - M1 * g * x1 - M0 * g * x0; % 势能

D = 1/2*c*(dx1 - dx0)^2;    % 损耗

W = F*x1 - F*x0;    % 外部做功

  

disp('K = ')

pretty(K)

  

disp('P = ')

pretty(P)

  

disp('D = ')

pretty(D)

  

disp('W = ')

pretty(W)

  

% 利用拉格朗日推运动方程

  

% 得拉格朗日函数

L = K - P;

disp('L = ')

pretty(L)

  

% 求运动方程

  

% 物体0的运动方程

Fi0_Lagrange  = diff(diff(L,dx0), t) - diff(L, x0);

disp('Fi0_Lagrange  = ')

pretty(Fi0_Lagrange )

  

F0 = -F + c * (dx1 - dx0);

% 解F

eq0 = F0 == Fi0_Lagrange;

F_sol = solve(eq0, F);

disp('F = ');

pretty(simplify(F_sol));
```
