什么是齐次变换？
	即不出现“原点位移”即加法。通过齐次变换等价坐标变换，可以将原来的坐标进行升维度，在思维空间进行齐次变换（乘法），利用四维空间的齐次变换描述三维空间的一般坐标变换。
为什么选择齐次变换？
		这样可以利用齐次变换的各种性质。在齐次坐标变换中，每次变换都是乘法，当多次变换的时候只需要进行乘法运算。并且平移，旋转，缩放，切变在思维空间中都表现为矩阵乘法。
		![[Pasted image 20260409105700.png]]	
		而在三维空间中的旋转+平移变换，面临多次变换的时候会出现加法和乘法的“套娃”，计算复杂。
		![[Pasted image 20260409105632.png]]
		【注】值得注意的是三位空间的坐标在四维空间中的w始终为1(按照人类世界的视角来看)即w = 1
		如，(2,0,0)T在四维空间的位置为(2,0,0,1)T

### 平移齐次坐标变换
在任何坐标变换，都可以在四维空间中仅用矩阵乘法来实现。因此引出平移齐次坐标变换（上文用的三维坐标变换的等价齐次变换形式）
平移齐次坐标变换可以实现以下作用：
	新位置坐标 = 旧位置 + 各个坐标的增量

## 通用旋转变换

即绕任意轴（非x,y,z）的旋转变换

### 通用旋转变换公式
![[Pasted image 20260411170225.png]]

| **符号**          | **含义**           | **说明**                                       |
| --------------- | ---------------- | -------------------------------------------- |
| $\mathbf{f}$    | **旋转轴向量**        | 必须为单位向量，即 $\sqrt{f_x^2 + f_y^2 + f_z^2} = 1$ |
| $f_x, f_y, f_z$ | **轴向量分量**        | 旋转轴在参考坐标系 $x, y, z$ 上的投影                     |
| $\theta$        | **绕f的旋转角度**      | 遵循右手定则，单位通常为弧度 (rad)                         |
| $c\theta$       | **余弦值**          | $\cos \theta$                                |
| $s\theta$       | **正弦值**          | $\sin \theta$                                |
| $vers\theta$    | **正矢 (Versine)** | $1 - \cos \theta$                            |

原理
	![[Pasted image 20260411170244.png]]
	转到{C}在{C}中旋转然后再回到原系。其中{C}的z轴与单位向量f对齐。因此就得到一个绕f轴旋转的通用旋转变换公式。
