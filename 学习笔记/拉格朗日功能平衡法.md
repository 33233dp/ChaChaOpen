``` matlab
syms g;

syms t;

% 对m1有

syms m1 theta1(t) d1;

dtheta1 = diff(theta1, t);

K1 = 1/2 * m1 * (d1 * dtheta1) ^ 2;

P1 = -m1 * g * d1 * cos(theta1);

% 对m2有：

syms d2 m2 theta2(t);

dtheta2 = diff(theta2, t);

x2 = d1*sin(theta1) + d2*sin(theta1+theta2);

y2 = -d1*cos(theta1) - d2 * cos(theta1+theta2);

dx2 = diff(x2, t);

dy2 = diff(y2, t);

  

K2 = 1/2 * m2 * (dx2 ^ 2 + dy2 ^ 2);

P2 = m2 * g * y2;

  

% 总能量

K = K1 + K2;

P = P1 + P2;

  

% 拉格朗日功能平衡法

  

%% 拉格朗日函数

L = K - P;

  

%% 求拉格朗日方程

%%% 微分

dL_theta1 = diff(L, theta1);

dL_theta2 = diff(L, theta2);

dL_dtheta1 = diff(L, dtheta1);

dL_dtheta2 = diff(L, dtheta2);

ddL_dtheta1 = diff(dL_dtheta1, t);

ddL_dtheta2 = diff(dL_dtheta2, t);

%%% 得到拉格朗日方程

F1_lagrange = ddL_dtheta1 - dL_theta1;

F2_lagrange = ddL_dtheta2 - dL_theta2;

disp('F1_lagrange = ');

pretty(simplify(F1_lagrange));

disp('F2_lagrange = ');

pretty(simplify(F2_lagrange));
```

也就是从复杂的：

$T_1=f(\theta_1,\theta_2,\dot\theta_1,\dot\theta_2,\ddot\theta_1,\ddot\theta_2)$  
$T_2=f(\theta_1,\theta_2,\dot\theta_1,\dot\theta_2,\ddot\theta_1,\ddot\theta_2)$  

整理成更有物理意义的形式：

$T = M(q)\ddot q + C(q,\dot q) + G(q)$  

其中：

$q=  \begin{bmatrix}  \theta_1\  \\ \theta_2  \end{bmatrix}$

---
### 1. (D_{ii})：有效惯量

比如：

$D_{11}=(m_1+m_2)d_1^2+m_2d_2^2+2m_2d_1d_2\cos\theta_2$  

它表示关节 1 看到的“等效转动惯量”。

也就是说，如果关节 1 有角加速度 ($\ddot\theta_1$)，那么它需要产生一部分力矩：

$D_{11}\ddot\theta_1$  

所以 (D_{11}) 类似于“质量”，只不过是在转动关节里的“惯量”。
>转动惯量：类似于质量
>平动：力 = 质量 X 加速度
>转动：力矩 = 转动惯量 X 角加速度
>转动惯量 = 质量 X 轴距的平方


---

### 2. (D_{ij})：耦合惯量

比如：

[  
$D_{12}=m_2d_2^2+m_2d_1d_2\cos\theta_2$  
]

它表示关节 2 的加速度会影响关节 1 的力矩。

也就是说，即使你只加速关节 2，关节 1 上也可能需要额外力矩：

[  
$D_{12}\ddot\theta_2$  
]

这就是机械臂多关节之间的耦合。

---

### 3. ($D_{ijk}\dot\theta_j^2$)：向心力项

比如类似：

[  
$-m_2d_1d_2\sin\theta_2\dot\theta_2^2$  
]

这种是速度平方项。

它表示某个关节转得越快，产生的向心惯性效应越大。
>向心加速度：质点在做圆周运动的时候，有一个指向圆形的向心加速度。$a_c​=rω^2$
>向心力：$F=m a_c$
>

---

### 4. ($D_{ijk}\dot\theta_j\dot\theta_k$)：科氏力项

比如：

$-2m_2d_1d_2\sin\theta_2\dot\theta_1\dot\theta_2$  

这种是两个速度相乘的项。

它表示两个关节同时运动时产生的耦合速度效应，通常叫科氏力项。

---

### 5. ($D_i$)：重力项

比如：

$D_1=(m_1+m_2)gd_1\sin\theta_1+m_2gd_2\sin(\theta_1+\theta_2)$  

它表示关节 1 为了抵抗重力需要提供的力矩。

---

## 这样做有三个好处：

第一，能看出每个力矩来源于哪里：惯性、耦合、向心、科氏、重力。

第二，方便编程实现。比如机器人控制里经常写成：

$\tau = M(q)\ddot q + C(q,\dot q)\dot q + G(q)$  

第三，方便做控制器设计。例如重力补偿、计算力矩控制、轨迹跟踪控制都会用这个结构。

---

**机器人关节力矩不是只和加速度有关，还和关节位置、速度、重力有关；从“数学公式”变成“工程上可用的动力学模型。
