# 快速排序

现在对33 10 15排序（倒序）

**确定基线条件**
- -  -
如果只有一个数，或者只是一个空数组，那么无需排序，则就是基线条件（退出递归的条件）


**确定缩小规模接近基线条件**

如果有两个数10,15那么只需要交换比10大的值就行

如果对33 10 15排序，先选取一个基准值如33。比33大的数放右边，小的放左边。则有10-15-33。现在将这个过程看作是一个函数即fastSort([33, 10, 15])。

如果有四个数[33, 10, 15, 4]那么可以选取一个基准值4。那么fastSort([33, 10, 15, 4])就是[4] ---[33,15,10]可见还可以对[33,15,10]进行排序同时可以发现，排序规模缩小了。因此可以利用这一点进行递归。即利用这一点缩小规模。

五个数[33, 10, 15, 4，6]的情况,若选取6为基准值，则排序为fastSort([4])---6----fastSort([33, 10, 15])

以此类推


以上过程实际上使用了归纳证明的方法，用于证明算法行之有效。

```cpp
#include <iostream>
#include <vector>

using namespace std;

vector<int> quickSort(vector<int> arr)
{
    // 基线条件：空数组或只有一个元素时，不需要再排序
    if (arr.size() <= 1)
    {
        return arr;
    }

    // 选择最后一个元素作为基准值
    int pivot = arr.back();
    vector<int> left;
    vector<int> right;

    // 倒序排列：比基准值大的放左边，比基准值小或等于的放右边
    for (int i = 0; i < arr.size() - 1; i++)
    {
        if (arr[i] > pivot)
        {
            left.push_back(arr[i]);
        }
        else
        {
            right.push_back(arr[i]);
        }
    }

    // 递归排序左右两边
    left = quickSort(left);
    right = quickSort(right);

    // 拼接结果：左边 + 基准值 + 右边
    left.push_back(pivot);
    left.insert(left.end(), right.begin(), right.end());

    return left;
}

int main()
{
    vector<int> arr = {33, 10, 15, 4, 6};
    vector<int> result = quickSort(arr);

    for (int num : result)
    {
        cout << num << " ";
    }
    cout << endl;

    return 0;
}

```
