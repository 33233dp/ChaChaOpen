给定一个字符串 `s` ，请你找出其中不含有重复字符的 **最长子串** 的长度。
- - - 
#### 示例 1:
输入：`s = "abcabcbb"` 
输出：`3` 
解释：因为无重复字符的最长子串是 `"abc"`，所以其长度为 3。注意 `"bca"` 和 `"cab"` 也是正确答案。
#### 示例 2: 
输入：`s = "bbbbb"` 输出：`1`
解释：因为无重复字符的最长子串是 `"b"`，所以其长度为 1。 
#### 示例 3: 
输入：`s = "pwwkew"` 
输出：`3` 
解释：因为无重复字符的最长子串是 `"wke"`，所以其长度为 3。
请注意，你的答案必须是 **子串** 的长度，`"pwke"` 是一个子序列，不是子串。
--- #### 提示:
- $0 \le \texttt{s.length} \le 5 * 10^4$ - `s` 由英文字母、数字、符号和空格组成
### 滑动窗口
- - -
**核心思想**
	👉 用一个“窗口”（区间）在数组/字符串上滑动  
	👉 通过移动左右边界，动态维护窗口内的状态
**两种常见类型**
	1. 固定窗口长度
	2. 可变窗口长度
**算法核心**
	- 双指针：
		👉 表示一个区间 `[left, right]`  
		👉 通过移动指针来“滑动窗口”
	- 变量更新
	- 区间合法维护
		while (不满足条件) {  
		    left++;  
		}

### 无重复字符串的最长子串
- - - 
``` cpp
class Solution { 
	public: int lengthOfLongestSubstring(string s) { 
		int len=s.size(); 
		int left=0; 
		int right=0;
		char hash[128]={0}; 
		int ret=0; 
		while(right<len) 
		{ 
			hash[s[right]]+=1;
			while(hash[s[right]]>1) 
				hash[s[left++]]--;
			ret=max(ret,right-left+1);
			right++; 
		}
		return ret; } 
};		
```
